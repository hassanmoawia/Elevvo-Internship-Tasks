# Elevvo-Internship-Tasks
Project Overview

This repository contains dashboards and analysis developed during my Data Analytics Internship at Elevvo.

The objective of this project was to transform raw operational and sales data into meaningful insights using modern business intelligence tools. The project demonstrates the end-to-end data analysis workflow including:

Data cleaning and preparation

SQL data modeling

KPI development

Interactive dashboard creation

The final result is a set of analytical dashboards designed to support data-driven decision making.

Tools & Technologies

This project was built using the following tools:

Microsoft SQL Server – data preparation and SQL views

Microsoft Power BI – data modeling and interactive dashboards

Microsoft Excel – data preprocessing and initial data exploration

Internship Task Objectives

During the internship at Elevvo, the main tasks were:

Transform raw data into structured datasets

Create SQL queries to extract business insights

Build Power BI dashboards for performance monitoring

Develop KPIs to evaluate business performance

Present insights through interactive visualizations

Dashboards Created
Sales Performance Dashboard

Analyzes revenue trends, product performance, and monthly sales patterns.

Customer Behavior Dashboard

Examines customer purchasing patterns, loyalty indicators, and satisfaction metrics.

KPI & Business Metrics Dashboard

Tracks key business indicators and performance metrics to support operational analysis.
